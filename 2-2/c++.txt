디폴트 매개변수를 갖는 함수 정의시
선언 후 나중에 함수를 정의하면 안됨
선언하면 바로 정의까지 같이 해야함

레퍼런스 : 한 변수의 또다른 이름(별명)

처음에 선언하면 반드시 초기화 할것
int &a; << x
int &a = b;  << O

한번 참조하면 변경 x

레퍼런스는 메모리 상에 존재하지 않을 수도 있다.
ex )  int a = 10;
      int &another_a = a; // another_a가 자리를 차지할 필요가 있을까?

참조자의 참조자는 없는 문법
(별명의 별명은 모순)

그냥 참조자 하나로 참조 ㄱ
